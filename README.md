# Twitch Enhancer

## Download

Click **[here](https://enhancer.at/#download)** to download extension.

---

## Install

1. `npm install`
2. `npm run build`
3. Add `build` directory as extension in your browser.
4. Refresh extension in your browser.

---

## Development

1. `npm install`
2. `npm run dev`
3. Add `build` directory as extension in your browser.
4. Refresh extension in your browser.
